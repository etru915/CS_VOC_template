Privacy Policy

Last updated: 2026-03-30

1. Overview

CS-VOC 상담 템플릿 (이하 "확장 프로그램")은 상담 분류 선택 시 미리 작성된 텍스트 템플릿을 상담 메모 영역에 자동으로 삽입하는 도구입니다.

2. Data Collection

이 확장 프로그램은 사용자의 개인정보를 수집, 저장, 전송하지 않습니다. 어떠한 사용자 데이터도 외부 서버로 전송되지 않습니다.

3. External Network Requests

이 확장 프로그램은 아래 목적으로만 외부 네트워크 요청을 수행합니다.

  • 요청 대상: https://raw.githubusercontent.com/etru915/CS_VOC_template/
  • 요청 내용: 상담 템플릿 텍스트 파일(.txt) 및 문의 유형 매핑 파일(.csv)
  • 요청 목적: 상담사가 선택한 문의 유형에 맞는 템플릿 텍스트를 화면에 표시
  • 전송 데이터: 없음. 파일 다운로드만 수행하며 사용자 정보는 전송하지 않습니다.

4. Permissions

이 확장 프로그램이 요청하는 권한과 그 이유는 다음과 같습니다.

  • host_permissions (raw.githubusercontent.com): 템플릿 텍스트 파일을 GitHub에서 불러오기 위해 필요합니다.
  • content_scripts (cs-voc.socarcorp.co.kr): 사내 상담 시스템 페이지에서만 동작하도록 제한되어 있습니다.
